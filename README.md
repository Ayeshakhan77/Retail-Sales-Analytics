# 🛒 Retail Sales Uplift Analysis & Customer Segmentation

## 📌 Project Overview  
This project focuses on analyzing retail transaction data to identify business growth opportunities through **customer segmentation, exploratory data analysis (EDA), and uplift testing**. The goal was to evaluate the impact of targeted strategies in **trial stores vs control stores** and measure their effect on overall sales performance.

The analysis resulted in a **7–10% sales uplift in trial stores**, driven by data-backed customer targeting and strategic experimentation.

---

## 🎯 Business Objective  
- Understand customer behavior through transaction data  
- Segment customers based on purchasing patterns  
- Evaluate the impact of store-level interventions  
- Measure sales uplift using **trial vs control store analysis**  
- Provide actionable business insights for revenue growth  

---

## 📊 Dataset Description  
The project used multiple datasets:

- **Sales Store Data** → Product-level and store-level sales information  
- **Transactions Data** → Detailed purchase records  
- **Purchase Data** → Customer buying behavior and frequency metrics  

These datasets were merged and transformed for analytical modeling.

---

## 🧠 Methodology  

### 1. Exploratory Data Analysis (EDA)
- Cleaned and preprocessed raw data  
- Identified sales trends, seasonality, and anomalies  
- Built visualizations using Python (Matplotlib / Seaborn)  
- Analyzed top-performing products and regions  

### 2. Customer Segmentation  
- Grouped customers based on purchasing frequency and value  
- Identified high-value and low-engagement segments  
- Derived behavioral patterns for targeted strategy design  

### 3. Trial vs Control Store Analysis  
- Selected comparable control stores for fair comparison  
- Applied uplift testing methodology  
- Measured performance differences before and after intervention  
- Quantified impact of marketing/strategy changes  

### 4. Impact Evaluation  
- Calculated sales uplift percentage  
- Validated results using comparative analysis  
- Confirmed **7–10% improvement in trial stores**

---

## 📈 Key Findings  
- Targeted customer strategies significantly improved store performance  
- Trial stores showed a consistent **7–10% increase in sales uplift**  
- High-value customer segments contributed disproportionately to revenue  
- Product-level insights helped identify best-performing categories  
- Data-driven decisions outperformed intuition-based strategies
